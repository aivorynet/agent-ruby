# Changelog

All notable changes to the AIVory Monitor Ruby Agent will be documented in this file.

This project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-02-16

### Added
- Automatic exception capture via at_exit hooks
- Thread.report_on_exception integration
- Manual exception capture API
- User and custom context enrichment
- WebSocket transport via websocket-client-simple
- Configurable sampling rate and capture depth
- Environment variable and programmatic configuration
- Ruby 2.7+ support
